==== Of Solitude And Stillness Version Hotfix (July 17) ====

We performed a hotfix on July 17 to address the following issues. Please be advised:

1. Fixed an issue where returning to the game's main interface could result in a black screen in some cases.

2. Fixed an issue where the blue dot indicator for claimable rewards was not displaying correctly on the [Event] - [Simulated Battlefield] interface.

3. Fixed an issue where the story could get stuck and fail to play properly during the [Journal of Promise: The Knower's Dilemma] stage in some cases.

4. Fixed an issue where [Helentine: Lacrimosa] could be unable to move properly in the Character Trial stage [The Knower's Dilemma] - [Silent Recollection] Challenge Stage 1, in some cases.

5. Fixed an issue where the background image on the [Of Solitude and Stillness] Floating Record interface did not fill the full screen when playing on mobile devices.


==== Of Solitude And Stillness Version Known Issues (July 17) ====

1. We're aware that [Helentine: Lacrimosa] can use the [Generic: Final Alternate Coating] during the [Journal of Promise: The Knower's Dilemma] stage. This will be fixed in a future version.

2. We're aware that the [Trial of Simulacrums] mode may, in some cases, get stuck during the tutorial sequence. Commandants can repeatedly tap on an empty area to force-skip the tutorial as a temporary workaround. This will be fixed in a future update.


Please refer above for the latest known issues and fixes. We apologize for any inconvenience this may cause.
